# Keyboard Shortcuts

## Navigation
| Key | Action |
|-----|--------|
| Arrow Keys | Navigate between streams |
| Z / Enter | Enter focused mode on selected stream |
| Esc | Exit focused mode / back |
| F | Toggle true fullscreen (hides browser chrome) |

## Playback
| Key | Action |
|-----|--------|
| Space | Play/Pause selected stream |
| M | Mute/unmute selected stream |
| N | Solo mode (mute all others) |
| P | Pause all streams |
| , | Seek back 10s |
| . | Seek forward 10s |
| < | Seek back 60s |
| > | Seek forward 60s |
| / | Random seek position |

## Stream Management
| Key | Action |
|-----|--------|
| X | Close/remove stream |
| R | Reload stream |
| D | Download stream |
| = | Remove duplicate streams |

## Likes & Slots
| Key | Action |
|-----|--------|
| L | Toggle like on selected stream |
| ` (backtick) | View liked streams only |
| 1-9 (tap) | Assign selected stream to slot |
| 1-9 (double-tap) | View streams in that slot |
| 0 | View all streams |

## Layout Modes
| Key | Action |
|-----|--------|
| T | Cycle Tetris mode (bin-packing layouts) |
| O | Toggle Coverflow mode |
| B | Bug Eye mode (toggle on/off, works in fullscreen) |
| Shift+B | Toggle Mosaic mode |

## UI
| Key | Action |
|-----|--------|
| H | Toggle header/toolbar |
| Shift+H | Toggle per-stream controls (clean mode) |
| V | Cycle through view filters |
| I | Toggle stream info overlay |
| S | Toggle streams panel |
| C | Copy selected stream URL |
| Shift+C | Copy all stream URLs |
| ? | Toggle shortcuts hint |

## Concept Summary

**Likes**: Binary "this is excellent" marker. Press `L` to toggle. Press `` ` `` to view only liked streams.

**Slots (1-9)**: Categorize related streams together. Single tap assigns, double tap views. Think of them as buckets for organizing your streams.

**Bug Eye**: Visual effect showing copies of the selected stream in a ring around the center. Press B to toggle on/off (works in fullscreen without exiting). Press Esc also exits.
