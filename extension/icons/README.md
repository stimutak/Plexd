# Extension Icons

Place PNG icons here:
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

Icon should be the Plexd logo - a simple "P" or grid pattern in purple (#6366f1).

For development, you can use placeholder images or generate them with:
- https://favicon.io/favicon-generator/
- Any image editor

The extension will work without icons (just shows a default puzzle piece).
